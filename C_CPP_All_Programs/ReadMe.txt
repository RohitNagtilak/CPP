|- Explicit Casting (Manual Casting)
   Explicit casting is performed manually by the programmer using casting operators.
   There are several casting operators in C++
   |
   |-CStyle casting 
   |-static_cast: Used for most ordinary type conversions that don't involve pointers or classes with base classes.
   |-dynamic_cast: Used for performing type-safe downcasting in polymorphic class hierarchies.
   |-const_cast: Used to add or remove the const qualifier from variables.
   |-reinterpret_cast: Used for low-level casting between unrelated types, often involving pointer
